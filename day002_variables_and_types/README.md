# Day 2: 변수와 기본 타입

## 개념 설명

Go 언어에서 변수는 **값을 저장하는 공간**입니다. Go는 정적 타입 언어로, 변수 선언 시 타입을 명시하거나 타입 추론을 통해 결정합니다. 변수는 **제로값**(zero value)으로 자동 초기화되며, 이는 타입에 따라 달라집니다 (예: int → 0, bool → false, string → "").

변수 선언 방식은 크게 세 가지입니다. 첫째, `var` 키워드로 선언하면 타입과 초기값을 명시할 수 있고, 초기값을 생략하면 제로값으로 초기화됩니다. 둘째, 함수 내에서 `:=`를 사용하면 타입을 생략하고 왼쪽 값에서 타입을 추론합니다. 셋째, `var` + 괄호로 여러 변수를 한 번에 선언할 수 있습니다 (구조체 등에서 유용).

기본 타입은 다음과 같습니다. `bool`, `string`, 정수형(`int`, `int8~64`, `uint`, `byte` = `uint8`), 부동소수점(`float32`, `float64`), 그리고 유니코드 코드포인트를 저장하는 `rune` (`int32` 별명)입니다. 특히 `rune`은 한글 등 멀티바이트 문자를 정확히 처리할 때 필수적입니다.

## 코드 설명

- **제로값 확인**: `var a int` 등으로 선언한 후 기본값이 0, false, "" 등임을 확인합니다.
- **초기값 선언**: `var x int = 42`처럼 타입과 함께 초기값 지정.
- **단축 선언 `:=`**: `name := "Alice"`는 `var name string = "Alice"`와 동치. 함수 내에서만 가능.
- **rune & byte**: `'한'`은 rune으로 자동 추론, `byte('A')`는 ASCII 범위에서 byte로 캐스팅.
- **UTF-8 처리 예시**: `[]rune("Go언어")`를 사용해 정확히 3글자로 인식. `len("Go언어")`는 바이트 수(9)가 아니라 rune 수(3)로 접근하려면 반드시 rune 변환 필요.
- **bool 활용**: 논리 연산 예시와 함께 빈 문자열이 false와 동치로 취급되는 조건문 예시 제공.

## 핵심 포인트

- **제로값은 타입별 기본 초기화 값**이며, 명시적 초기화 없이도 안전하게 사용 가능.
- **`var` vs `:=`**: 전역 변수나 명시적 타입 선언은 `var`, 지역 변수에서 타입 추론이 가능하면 `:=`를 권장.
- **`byte`는 ASCII용**, **`rune`은 유니코드용**으로 구분. 다국어(특히 한국어) 처리 시 rune 필수.
- **`string`은 불변**, **`[]rune`으로 변환해 글자 단위 조작 가능**.
