# Day 7: if / else

## 개념 설명

Go 언어에서 `if` 문은 조건부 실행을 가능하게 해주는 핵심 제어 흐름 구조입니다. `if` 문은 조건식이 `true`일 때 특정 블록을 실행하고, `false`라면 무시합니다. `else` 키워드는 조건이 거짓일 때 실행할 블록을 정의합니다.

Go의 `if` 문은 특별히 *초기화 구문*(Initialization Statement)을 지원합니다. 문법적으로 `if v := f(); v < 0 { ... }`와 같이, 조건 검사 이전에 변수를 선언하고 초기화한 후 바로 사용할 수 있습니다. 이 변수는 `if` 블록과 그에 딸린 `else` 블록 전체에서 유효합니다.

Go는 중괄호 `{}`를 필수로 요구하며, 조건식을 괄호로 감싸지 않습니다 (`if (x > 0)` → `x > 0` 가능하지만 권장되지 않음). 또한 `else if`를 체이닝하여 다중 조건을 간결하게 처리할 수 있습니다. `else` 블록은 동일한 범위(same scope)에서 선언된 `if` 초기화 변수를 접근할 수 있습니다.

## 코드 설명

1. **기본 `if` 문**: `age >= 18` 조건을 검사하여 투표 가능 여부를 판단합니다.
2. **`if / else` 쌍**: 나이에 따라 술 마실 수 있는지 여부를 판별합니다.
3. **`if / else if / else` 체인**: 점수에 따라 학점(A~D)을 분기 처리합니다.
4. **`if` 초기화 구문**: `if x := result * 2; x > 15 { ... }`에서 `x`는 단일 조건 검사 이전에 선언되고 계산됩니다.
5. **`else` 블록 내 스코프**: `y`는 `if`와 `else` 모두에서 사용 가능하므로, 조건에 따라 다른 메시지를 출력합니다.
6. **`if` + `else` 조합 예시**: 상태 문자열을 검사하여 DB 연결 결과를 보여줍니다.
7. **단순 조건 표현 패턴**: `0`, 양수, 음수를 각각 검사하여 수의 부호를 판단합니다.

## 핵심 포인트

- **`if` 초기화 구문**: `if v := expr; condition` 형태로, 조건식과 변수 초기화를 한 번에 처리
- **블록 스코프**: 초기화 변수는 `if`, `else`, `else if` 블록 전체에서 유효
- **`else if` 체이닝**: 다중 조건을 깔끔하게 분기 가능
- **중괄호 필수**: 조건식에 괄호는 선택이나, `{}`는 반드시 있어야 함
- **`else`는 `if`와 같은 블록 내에서만 가능** — 예: `if { ... } else { ... }`
