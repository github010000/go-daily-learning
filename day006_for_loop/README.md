# Day 6: for 반복문

## 개념 설명

Go 언어는 유일한 반복문으로 `for`를 제공합니다. C 스타일의 `while` 또는 `do-while`이 없지만, `for` 문을 활용해 동일한 동작을 표현할 수 있습니다.

기본 `for` 문은 초기화, 조건, 증감 표현식을 괄호 안에 콜론으로 구분해 작성합니다. 예: `for i := 0; i < n; i++`

조건만 있는 `for` 문은 `while`과 동일한 역할을 합니다. 괄호 안에 조건만 주면, 표현식이 `true`인 동안 반복됩니다. 이는 `for condition { ... }` 형태로 사용됩니다.

무한 루프는 조건을 생략한 채 `for { ... }`로 작성합니다. 반드시 `break` 또는 `return`을 통해 루프를 탈출해야 하며, 무한 루프는 이벤트 루프나 서버 대기 상태 등에서 자주 사용됩니다.

`range` 없이 인덱스를 직접 관리해 반복하는 것도 가능하며, 이는 컬렉션 크기나 조건부 처리가 필요한 경우 유용합니다.

## 코드 설명

- **기본 for**: `i := 0`, `i < 5`, `i++`로 0~4를 출력합니다.
- **조건만 있는 for**: `count > 0` 조건을 반복하면서 `count`를 감소시켜 카운트다운 효과를 구현합니다.
- **무한 루프**: `for {}`로 무한히 실행하되, `x >= 2` 조건에서 `break`로 탈출합니다.
- **index 기반 순회**: `for i := 0; i < len(numbers); i++`로 슬라이스 인덱스를 직접 사용해 각 요소에 접근합니다.
- **조건 기반 합계 계산**: `sum < 25`인 동안 반복하며 `numbers`의 합계를 누적합니다. 탈출 후 최종 합계를 출력합니다.

## 핵심 포인트

- Go에는 `for` 하나만으로 모든 반복을 처리합니다.
- `for condition {}`은 `while` 대용으로 사용 가능합니다.
- 무한 루프는 조건 없이 `for {}`로 작성되며, `break`로 반드시 탈출해야 합니다.
- `range` 대신 수동 인덱스 관리(`for i := 0; ...`)도 자유롭게 가능합니다.
- 반복 중 조건에 따라 동적으로 탈출하는 로직을 설계할 수 있습니다.
