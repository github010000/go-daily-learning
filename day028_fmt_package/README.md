# Day 28: fmt 패키지 심화

## 개념 설명
Go 언어의 `fmt` 패키지는 표준 입출력과 포맷팅을 담당하는 핵심 패키지입니다. 이번 학습에서는 문자열을 포맷팅하는 `Sprintf`, `Fprintf`, `Errorf` 함수와 데이터 형식을 지정하는 포맷 동사(Format Verbs), 그리고 입력을 처리하는 `Scanf` 함수에 대해 다룹니다.

`Sprintf`는 포맷팅된 문자열을 반환하여 변수에 저장하거나 다른 함수에 전달할 때 유용합니다. 반면, `Fprintf`는 파일을 포함한 `io.Writer` 인터페이스를 구현한 모든 대상(표준 출력, 파일 등)으로 데이터를 쓸 수 있게 해줍니다. `Errorf`는 오류 메시지를 생성하여 반환하는데, 에러를 반환하는 함수 내부에서 유용하게 쓰입니다.

데이터를 출력할 때 `%v`, `%+v`, `%#v` 등 다양한 포맷 동사를 사용하면 구조체와 같은 복잡한 데이터 타입을 직관적으로 확인할 수 있습니다. 특히 `%+v`는 구조체의 필드 이름을 함께 출력하여 디버깅에 큰 도움이 됩니다. 또한 `%T`는 타입을, `%q`는 따옴표 처리를 도와줍니다.

입력 처리의 경우 `fmt.Scanf`를 사용하여 표준입력에서 데이터를 읽고 변수에 저장할 수 있습니다. 하지만 `Scanf`는 입력 형식과 개수가 정확히 일치해야 하므로, 실제 프로덕션 환경에서는 `bufio.Scanner`나 `os.Stdin.Read` 등을 사용하는 것이 더 안전하고 권장됩니다.

## 코드 설명
1. **Sprintf 예제**: `fmt.Sprintf`를 사용하여 문자열, 정수, 기본 값 등을 포맷팅한 결과를 문자열 변수에 저장합니다. `%05d`와 `%x`와 같은 서식을 사용하여 정수 패딩과 16진수 변환도 보여줍니다.
2. **Fprintf 예제**: `fmt.Fprintf`를 사용하여 `os.Stdout`(표준 출력)으로 데이터를 직접 출력합니다. 이는 파일에 쓰거나 커스텀 Writer로도 확장 가능합니다.
3. **Errorf 예제**: `fmt.Errorf`를 사용하여 오류 메시지를 생성하고 반환합니다.
4. **포맷 동사 예제**: `SampleUser` 구조체를 정의하고 `%v`, `%+v`, `%#v`, `%T`, `%q`, `%x` 등 다양한 동사를 적용하여 출력 결과를 비교합니다.
5. **Scanf 예제**: `fmt.Scanf`를 사용하여 사용자에게서 이름과 나이를 입력받습니다. 입력 형식이 `%s %d`와 일치하지 않으면 에러가 반환되므로, 예외 처리를 포함하여 설명합니다.

## 핵심 포인트
- `Sprintf`: 포맷팅된 문자열을 **반환**합니다.
- `Fprintf`: `io.Writer`(표준출력, 파일 등)로 **출력**합니다.
- `Errorf`: 오류 메시지를 생성하여 **반환**합니다.
- `%+v`: 구조체의 필드 이름을 포함하여 출력합니다 (디버깅에 유용).
- `%T`: 데이터의 타입을 출력합니다.
